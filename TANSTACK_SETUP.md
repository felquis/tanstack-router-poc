# TanStack Router Setup - Performance Optimized

## ✅ Setup Complete

Your TanStack Router POC is configured with **static-first approach**, **route-level code splitting**, **link prefetching**, and **optimized build**.

## 🚀 Key Features Implemented

### 1. Route-Level Code Splitting
- Each route loads on demand in separate chunks
- `vendor.js` - React, TanStack libraries (~74KB gzip)
- `routes.js` - All route components (~41KB gzip)
- `index.js` - Main app bundle (~0.69KB gzip)

### 2. Link Prefetching
```tsx
// Configured in App.tsx with defaultPreload: 'intent'
// Links prefetch automatically on hover/mouse enter
<Link to="/about">About</Link>  // Prefetches when you hover
```

### 3. Lazy Route Components
Each route uses lazy-loaded components:
```tsx
export const Route = createFileRoute('/')({
  component: LazyIndexComponent,  // Lazy-loaded
})
```

### 4. Static-First Architecture
- No SSR setup (client-side only)
- All components are React components
- Client-side loaders when needed
- Fast initial load on static hosting

## 🌐 Deployment to Cloudflare Pages

### Method 1: Dashboard (Easiest)
1. Push to GitHub repo
2. Go to **Cloudflare Dashboard** → **Pages** → **Create project**
3. Connect your repo
4. Build settings:
   - **Build command**: `npm run build`
   - **Output directory**: `dist`
5. Deploy

### Method 2: CLI
```bash
# Install Wrangler
npm install -g wrangler

# Build and deploy
npm run build
wrangler pages deploy dist
```

### Method 3: Workers Static Asset Serving (Advanced)
Once on Pages, you can optionally upgrade to Workers for:
- Custom edge logic
- Direct asset serving from Workers
- Cached responses

**Start with Pages first** — it's the easiest and provides global edge delivery.

## 📱 Optional: Add Client-Side Loaders

When you need to fetch data in routes:

```tsx
// src/routes/about.tsx
export const Route = createFileRoute('/about')({
  loader: async () => {
    // Fetch data before component renders
    const data = await fetch('/api/data').then(r => r.json())
    return { data }
  },
  component: LazyAboutComponent,
})

function LazyAboutComponent() {
  const { data } = Route.useLoaderData()
  return <div>{JSON.stringify(data)}</div>
}
```
